## 4. 技術スタック

### フロントエンド
- **Framework**: Next.js 14（App Router）
- **スタイリング**: Tailwind CSS + CSS Modules
- **アニメーション**: Framer Motion
- **状態管理**: Zustand（必要に応じて）

### ビルド・デプロイ
- **ホスティング**: Vercel
- **画像最適化**: Next.js Image Component
- **動画**: MP4形式（lazy loading実装）

### パフォーマンス最適化
- ISR（Incremental Static Regeneration）
- 画像のWebP変換
- フォントの事前読み込み
- Critical CSSのインライン化
